函数模板

template <typename T> [constexpr/inline] // []表示可选 可以将该函数声明为内联函数 或者常量表达式
int compare(const T& v1, const T& v2) {
    if (v1 < v2) return -1;
    if (v2 < v1) return 1;
    return 0;
}
  1. 模板定义 以template关键字开始
  2. < >是模板参数列表，参数分为类型参数和非类型参数，其中类型参数前使用关键字typename 或 class, 非类型参数前则使用特定的类型名
  3. 实例化: 仅在实例化时，编译器才会通过给定参数生成代码 在实例化时， 编译器通过具体给出的实参来推断形参的类型或者是值。在实例化时，非类型参数对应的实参必须是常量表达式
  4. 原则: 函数参数都是const的引用 函数体内条件判断只是用<
  5. 头文件: 模板的头文件通常既包括声明 也包括定义

类模板

template <template_parameter_list> 
class class_name {
	ret_type function_name(parameters) { }
};
template <template_parameter_list> 
ret_type class_name<template_parameter_list>::function_name(parameters) {
    
}
// eg:
template <typename T> 
class Blob {
public:
    T& back();

private:    
    std::shared_ptr<std::vector<T>> data;
};
template <typename T>
T& Blob<T>::back() {
    return data->back();
}

  1. 友元声明

    1. 将相同类型实例化的类声明为友元

      template <typename> class BlobPtr;
template <typename T> class Blob {
    friend class BlobPtr<T>; 
}

    2. 将类的所有实例都声明为本类的每个实例的友元

      template <typename T> class C2 {
	// 对于C2的每一个实例来说 Pal2的每一个实例都是他的友元
	template <typename X> friend class Pal2;
}

    3. 将类声明为本类的每个实例的友元

      template <typename T> class C2 {
    // 对于C2的每一个实例来说Pal3都是他的友元
	firend class Pal3;
}

  2. 类型别名

    template <typename T> using twin = pair<T, T>
twin<int> win_loss; // pair<int, int>
twin<string> partNo; // pair<string, string>

  3. static成员

    template <typename T> class Foo {
private:
    // 所有的Foo<X>共享相同的ctr Foo<T>与Foo<X>的ctr不同
    static std::size_t ctr;
    // 上述ctr也可以定义成以下形式
    template <typename T>
    size_t Foo<T>::btr = 0;
}
